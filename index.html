Workflow CSV2WikiChart

Basic Workflow of CSV2WikiChart is:
  • Load a CSV File that should be converted into a Chart for Wikiversity or Wikipedia.
  • Check data in a graph with the library MorrisJS as preview
  • Edit CSV data or the settings for the graph in Wikiversity or Wikipedia and check again the preview of the chart

About

WebApp - CSV2WikiChart
1.1.11

AppLSAC

This is a JSON editor with Load, Save and Print features running completely in the browser without submission of loaded data to remote servers.
- see CSV2Chart on Wikiversity.
  • Wiki Template: The JSON Editor exports to the Wikipedia/Wikiversity Template:Graph:Chart.
  • JSON-Editor App The underlying JSON-Editor was used for this repository csv2wikichart_app on GitHub. Editor layout for input is configured by a JSON Schema W to generate tailored JSON
  • Template for Graphs: The syntax for the output is defined by the Wikipedia/Wikiversity Template - Graph:Chart. The Handlebars4Code template generates the Wiki-Graph output with the library Handlebars4Code. The output of CSV2WikiChart WebApp uses the JSON in the JSON Editor to populate the Handlebars4Code-Template tpl/template4json_tpl.js (see also Handelbars4Code Demo).
  • App Menu: The menu was created with the JSONEditor4Menu as a web-based interface to create a Hamburger Menu App. Editor layout for input is configured by a JSON Schema W to generate tailored JSON
  • used icons for the menu are a icons from JQuery Mobile 1.4.5 Icons and from Based on WikiCommons Icons: Icons for Menu: The used icons for the menu are a icons from JQuery Mobile 1.4.5 Icons and from WikiCommons. For icons and licensing detail see README_Icons4Menu.html.
This is an AppLSAC. The application was create with JSONEditor4Menu. See also GitLab repository for source code of JSONEditor4Menu
